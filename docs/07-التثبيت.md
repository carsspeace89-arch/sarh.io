| المكوّن | الإصدار الأدنى |
|---------|---------------|

| Apache | 2.4+ (مع mod_rewrite) |

## التثبيت المحلي (Laragon)

### 1. وضع الملفات

```
C:\laragon\www\LINK3\attendance-system\
```

### 2. إعداد config.php

افتح `includes/config.php` وعدّل:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');          // كلمة مرور MySQL
define('DB_NAME', 'attendance_system');
define('SITE_URL', 'http://localhost/LINK3/attendance-system');
```

### 3. إنشاء قاعدة البيانات

```bash
# من سطر الأوامر (Laragon)
mysql -u root -e "CREATE DATABASE attendance_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```

### 4. تشغيل المثبّت

1. افتح `.htaccess` وعلّق على السطر:
   ```apache
   # Deny from all   (ضمن <Files "install.php">)
   ```
2. افتح `http://localhost/LINK3/attendance-system/install.php`
3. اضغط "تثبيت" ← تُنشأ جميع الجداول ويُضاف المشرف الافتراضي
4. أعد تفعيل الحجب في `.htaccess`

## النشر على الاستضافة (Hosting)

### إعدادات config.php للإنتاج

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'cpanel_user_db');
define('DB_PASS', 'strong_password');
define('DB_NAME', 'cpanel_user_attendance');
define('SITE_URL', 'https://yourdomain.com/attendance-system');
```

### تفعيل HTTPS في .htaccess

أزل التعليق عن سطور إعادة التوجيه:
```apache
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [R=301,L]
```

### إعداد session.cookie_secure

```php
ini_set('session.cookie_secure', 1);   // HTTPS فقط
```

### قواعد .htaccess الأمنية المُفعَّلة

```apache
# منع الوصول لملفات PHP في includes/
<Directory "includes">
    Order Allow,Deny
    Deny from all
</Directory>

# حجب install.php بعد التثبيت
<Files "install.php">
    Order Allow,Deny
    Deny from all
</Files>

# منع تصفح المجلدات
Options -Indexes

# حماية .htaccess نفسه
<Files ".htaccess">
    Order Allow,Deny
    Deny from all
</Files>
```

## بعد التثبيت — خطوات إلزامية

1. **غيّر كلمة المرور** من الإعدادات ← تبويب الأمان
2. **حدّد موقع العمل** من الإعدادات ← تبويب الموقع الجغرافي
3. **اضبط أوقات الدوام** من الإعدادات ← تبويب أوقات الدوام
4. **أضف الموظفين** من صفحة إدارة الموظفين
5. **أرسل الروابط** لكل موظف عبر واتساب

## النسخ الاحتياطي

### نسخ قاعدة البيانات يدوياً

```bash
mysqldump -u root -p attendance_system > backup_$(date +%Y%m%d).sql
```

### استعادة نسخة احتياطية

```bash
mysql -u root -p attendance_system < backup_20260302.sql
```

## تحديث النظام

عند تلقّي ملفات محدثة:
1. أنسخ احتياطياً قاعدة البيانات أولاً
2. استبدل الملفات (**عدا** `includes/config.php`)
3. إذا وُجد ملف `update.php` ← شغّله مرة واحدة ثم احجبه
