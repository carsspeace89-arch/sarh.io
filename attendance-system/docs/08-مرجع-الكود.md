# مرجع الكود — المكتبات والدوال

## includes/config.php — الإعدادات الأساسية

### الثوابت المعرّفة

| الثابت | القيمة الافتراضية | الوصف |
|--------|------------------|-------|
| `DB_HOST` | `localhost` | عنوان خادم MySQL |
| `DB_USER` | `root` | مستخدم قاعدة البيانات |
| `DB_PASS` | `''` | كلمة المرور |
| `DB_NAME` | `attendance_system` | اسم قاعدة البيانات |
| `DB_CHARSET` | `utf8mb4` | ترميز الاتصال |
| `SITE_URL` | `http://localhost/LINK3/attendance-system` | رابط الموقع (بدون `/` في النهاية) |
| `SITE_NAME` | `نظام الحضور والانصراف` | اسم الموقع |

## includes/db.php — الاتصال بقاعدة البيانات

### نمط Singleton

```php
// الاستخدام في أي مكان:
$result = db()->prepare("SELECT ...");
```

- كائن PDO واحد لكل طلب HTTP (Singleton Pattern)
- `PDO::ATTR_ERRMODE = ERRMODE_EXCEPTION` — يرمي استثناءات عند الخطأ
- `SET NAMES utf8mb4` — يضمن ترميز عربي صحيح

## includes/functions.php — الدوال العامة

### دوال قاعدة البيانات والإعدادات

```php
// جلب إعداد واحد (مع caching)
getSystemSetting(string $key, string $default = ''): string

// حفظ إعداد (يُبطل الـ cache تلقائياً)
setSystemSetting(string $key, string $value): void

// تحميل كل الإعدادات دفعة واحدة (تجنب N+1 queries)
loadAllSettings(bool $refresh = false): array
```

### دوال الموظفين والحضور

```php
// جلب موظف عبر token
getEmployeeByToken(string $token): ?array

// حساب المسافة بالمتر (Haversine)
calculateDistance(float $lat1, float $lon1, float $lat2, float $lon2): float

// التحقق من الجيوفينس — يعيد ['allowed', 'distance', 'radius', 'message']
isWithinGeofence(float $empLat, float $empLon): array

// تسجيل حضور/انصراف — يعيد ['success', 'message']
recordAttendance(int $employeeId, string $type, float $lat, float $lon, float $accuracy): array

// تحقق من تكرار التسجيل
hasRecentAttendance(int $employeeId, string $type, int $minutes = 5): bool
```

### دوال الأمان

```php
sanitize(string $value): string          // XSS protection
generateCsrfToken(): string              // إنشاء CSRF token
verifyCsrfToken(string $token): bool     // التحقق من CSRF
generateUniqueToken(): string            // توليد رابط موظف (64 hex)
getClientIP(): string                    // IP حقيقي (حماية IP Spoofing)
```

### دوال المساعدة

```php
jsonResponse(array $data, int $code = 200): void  // إرجاع JSON وإنهاء
generateWhatsAppLink(string $phone, string $token): string
arabicDateTime(string $datetime): string
getTodayStats(): array  // ['checked_in', 'checked_out', 'total_employees']
```

## employee/attendance.php — صفحة الموظف

### متغيرات PHP الرئيسية

```php
$employee       // بيانات الموظف من DB
$todayStatus    // 'none' | 'checked_in' | 'checked_out'
$checkInTime    // وقت الدخول اليوم (أو null)
$checkOutTime   // وقت الانصراف اليوم (أو null)
$workLat        // خط عرض موقع العمل
$workLon        // خط طول موقع العمل
$geofenceRadius // نصف القطر المسموح (متر)
$showCheckoutButton // bool: هل يُعرض زر الانصراف
$jsConfig       // JSON يُمرَّر لـ JavaScript
```
